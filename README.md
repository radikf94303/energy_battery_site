# energy_battery_site
Site layout webpage that allows users to mockup the build of  materials and site layout of an Industrial Energy Battery site

A full-stack industrial energy storage site configurator and layout planner.

## Quick Start

```bash
npm start
```

This single command:
1. Installs all dependencies (frontend + backend)
2. Starts the **API backend** on http://0.0.0.0:3001
3. Starts the **React frontend** on http://0.0.0.0:1000
